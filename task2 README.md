1.  Сохраните код сервера в файл `server.py`.
2.  Сохраните код клиента в файл `client.py`.
3.  Запустите сначала серверную часть в терминале/командной строке:
    python server.py
    Вы должны увидеть сообщение: `"Сервер запущен и ожидает подключений..."`.
4.  Затем в другом окне терминала запустите клиентскую часть:
    python client.py
